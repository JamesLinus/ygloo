Build signature base string for oAuth, format: urlencode(METHOD)&urlencode(BASEURL)&urlencode(PARAMS)

Gotchas:

  1. Parameter values are double encoded in the end, because "value" in oAuth lingo is "value as it is sent to the server" and values are sent to the server URL encoded.
