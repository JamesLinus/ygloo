Detailed Description

Vbitmap drawing primitives.

Vbitmap abstract type for images.

This module provides API for creating and edditing Vbitmaps which is an abstract type for an image into Ymagine

Constructs a Vbitmap whose buffer's memory is controlled by the libraryOn creation, the Vbitmap as a null size. Caller should callVbitmapResize() to set the size of the Vbitmap, and have it dynamically allocate the matching pixel buffer.

When the user is done with the Vbitmap, must call VbitmapRelease() to release the Vbitmap.

unsigned char* VbitmapRegionBuffer ( Vbitmap *  vbitmap)

Returns the buffer that contains the data for active region of the imageBefore accessing the buffer, vbitmap must first be locked with VbitmapLock(). Afterwards, vbitmap must be unlocked with VbitmapUnlock().

If there isn't any active region, it returns the buffer of the original image. It returns NULL, if the region is empty and/or doesn't intersect with the Vbitmap full buffer.
