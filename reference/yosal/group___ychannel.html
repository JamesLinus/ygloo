Detailed Description

Stackable input/output channel that can read from or write to different types of data.

For example a Ychannel could be backed by a file descriptor, a memory buffer or basically anything that the Ychannel callbacks can be implemented for. Refer to src/io/engine/ for example implementations of these functions. An implementation of all 4 callbacks is called an engine and the data they operate to is referred to as engine data.
