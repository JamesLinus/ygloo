Detailed Description

Hashmap data structure.

Portable Hashmap implementation in C that supports arbitrary (void*) keys and values.

Lock a Yhashmap. This function blocks until the lock can be acquired and does not attempt to prevent deadlocks. A Yhashmap never locks or unlocks itself. If a Yhashmap is shared between threads, each thread has to call Yhashmap_lock before interacting with the Yhashmap.
